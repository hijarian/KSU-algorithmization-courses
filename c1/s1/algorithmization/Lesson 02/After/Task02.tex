\textbf{Задача:}
Составить циклический алгоритм и программу для вычисления результата по формуле.
Для проверки программы задать $X = 0,5; n = 20$.

\begin{equation}
e^{\sqrt{x/n}} \sum_{k=1}^n \frac{|x-k|^2 \sqrt{e^{(k - 1)}} }{\ln( 2 + x^k + x^{2k + 1} ) }
\end{equation}
